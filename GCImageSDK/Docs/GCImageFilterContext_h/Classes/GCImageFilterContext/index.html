GCImageFilterContext

Superclass:
GCFilterContext
Declared In:
GCImageFilterContext.h

Introduction

GCImageFilterContext provides a context for query all required info to create GCImageFilter instance or set parameters of the GCImageFilter object, etc.

Discussion

There're two approaches provided in GCImageFilterContext to access the info about image filters,

  1. Get all image filter category info, and all image filter info in each category with GCImageFilterCategoryInfo and GCImageFilterInfo.

  2. Get a image filter category name or a image filter name first, and then retrieve other info by the name.
